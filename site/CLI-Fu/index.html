    Introduction¶

    This document serves as an accompanying guide/index to the command-line tools discussed in the notebooks. Here we have presented the relevant functions/verbs from command-line-tools that work with tabular data in the context of data-analysis tasks.

    00 - Import, Inspect¶

    conversion¶

    • from/to csv
    in2csv, csv2json
csvtk csv2tab, space2tab, tab2csv
xsv fmt
mlr cat

## compressed data?
mlr --prepipe

    display¶

    head, tail
csvlook
csvtk pretty
xsv table
mlr head, tail

    count¶

    • rows, columns
    wc
xsv count

    types¶

    • detect types, conversion
    mlr put is_*
mlr put boolean, int, float, string

    column names¶

    csvcut -n
xsv headers
csvtk headers
mlr label

    01 - Subset​¶

    rename¶

    • one/many columns
    csvtk rename, rename2
mlr rename

    index¶

    • create row names/indentifiers
    nl
xsv index

    subset columns¶

    • select/exclude cols
    cut
csvcut
csvtk select
xsv select
mlr cut
mlr having-fields

    subset rows¶

    • select/exclude rows
    cut
csvgrep
csvtk filter, filter2, grep
xsv search
mlr filter

    sample¶

    • with (bootstrap) or without replacement (permutation)
    shuf
csvtk sample
xsv sample
mlr bootstrap, sample, shuffle, decimate

    split¶

    • large file into smaller files
    split
csplit
xsv split

    02-Clean​¶

    missing values¶

    awk
csvstat --nulls
mlr put is_null, is_not_null

    duplicates¶

    mlr repeat # create dups
datamash rmdup

    03-Mutate​¶

    mutate¶

    • create/drop rows, cols
    awk
mlr put
csvtk mutate

    format¶

    • numerical formatting
    numfmt
datamash round, ceil, floor, trunc, frac

    time conversions¶

    • from/to epoch
    mlr put with strftime, strptime
mlr sec2gmt, sec2gmtdate

    functions¶

    • apply, map
    awk
mlr put

    discretize¶

    • cut numerics into categoricals
    datamash bin
mlr histogram --nbins

    reshape¶

    • long/wide to wide/long
    paste
csvtk transpose
datamash transpose

# pandas-like reshape
mlr reshape

    04-Merge/Join/Concat¶

    join¶

    • merge tables
    join
csvtk join
xsv join

    concat¶

    • append/concat tables
    cat
csvstack
xsv cat

# concat when cols are not same
mlr unsparsify

    compare, intersect¶

    • rows in A & B, rows in A not in B etc.
    comm
csvtk intersect

    05-Explore¶

    aggregate¶

    • group-by, pivot
    datamash
mlr

    sort¶

    sort
csvsort

    uniques¶

    uniq
csvtk uniq

    frequencies¶

    uniq -c
csvtk freq
xsv frequency
mlr top
mlr least-frequent, most-frequent
mlr fraction # convert frequencies to percentages

    crosstabs¶

    datamash crosstab

    06-Analyze/Visualize​¶

    univariate¶

    • mean/stddev, median, percentiles, skewness/kurtosis, mode, min/max
    csvstat
csvtk stats, stats2
xsv stats
mlr stats1, stats2
datamash

    bivariate¶

    • correlation/covariance, regression, r-squared
    mlr stats2

    visualize¶

    • histograms, scatterplots
    csvtk plot
mlr bar

    07-Advanced¶

    generate¶

    • random data
    seq, shuf, pr
mlr seqgen
mlr put "urand(), urandint()"

    query¶

    • run sql queries
    csvsql
q -H -d, """query"""

# generate a CREATE TABLE query for your csv
# super useful when pushing data into a local db (mysql, postgresql etc.)
csvsql -i sqlite joined.csv
